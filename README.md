# WeekendGo · 那我走 — 菲律宾徒步 3D 地图

Philippines National Hiking Map — 60 mountains across Luzon, Visayas and Mindanao on an interactive 3D terrain map.

**Live site:** https://jyb635050-ai.github.io/weekendgo-map/

## Features / 功能

- 🏔️ 3D satellite terrain map (MapLibre GL JS + AWS Terrain Tiles + Esri World Imagery) — drag to rotate 360°, tilt, orbit
- 📍 60 hikeable mountains with difficulty-coded markers (PinoyMountaineer 1/9–9/9 scale)
- 📖 AllTrails-style guides per mountain: elevation, difficulty, trail class, time to summit, jump-off, tips, best season, sources
- 🌐 中文 / English bilingual, 🌙 dark / light mode, 🔍 search + region & difficulty filters
- 🔗 Deep links: `?m=pulag` opens a mountain directly; `?nohero=1` skips the intro

## Data sources / 数据来源

Mountain specs distilled from [Pinoy Mountaineer](https://www.pinoymountaineer.com/) guide pages
(difficulty, trail class, coordinates, itineraries), with status checks from PHIVOLCS and news sources.
Descriptions are original bilingual writing. Fields we could not verify are marked "待确认 / TBC".

Made by **WeekendGo（那我走）** hiking collective 🇵🇭
